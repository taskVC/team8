Checkout (Old vs New)

\#1. How to view commits:

Use git log --oneline to see the history.



\#2. Switching to Old Version:

Use git checkout <commit\_hash>. 

This allows you to view the project files as they were in the past.



\#3. Returning to New Version:

Use git checkout task-checkout to return to your current work.
